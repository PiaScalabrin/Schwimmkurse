Starte das Programm mit menu.py 

Person.py = Teilnehmer und Prüfer 

Verwaltung.py = stellt die Logik zur Verwaltung der Daten bereit.

Abzeichen = enthält die Abzeichen

Rettungsschwimmen.py ist der ganze Code
